; These are working labels for the original joystick build.  The file is still
; a plain disassembly dump, so labels below are guideposts rather than symbols
; consumed by an assembler.
;
; Important zero-page state touched by the input path:
;   $1c = fire latch/current fire state, set from joystick bit 4
;   $1d = previous/edge fire state used when fire transitions
;   $1e = horizontal input command consumed by later player movement code
;         joystick writes $80 for left, $01 for right, $00 for neutral;
;         movement code tests it as zero/nonzero and signed.
;   $c9 = joystick mask built earlier from keyboard/control selection code
;
; C64 input registers used here:
;   $dc00/$dc01 = CIA #1 keyboard/joystick matrix ports
;   $dc02/$dc03 = CIA #1 data direction registers for ports A/B
;   $d418      = SID volume/filter register; also selects paddle lines when
;                the patched version wants to read $d419/$d41a.
